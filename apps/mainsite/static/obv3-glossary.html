Edubadges OpenBadge version 3.0 Glossary

This document provides definitions for terms used in the Edubadges OpenBadge version 3.0 specification. It explains the extensions and differences with the published OpenBadge Specification.

RegularAchievement

A badge class for acquired knowledge and skills that are part of the curriculum of a recognized education program. This badge class is associated with ECTS/SBU.

ECTS
European Credit Transfer and Accumulation System
SBU
Study Burden Unit

MicroCredentialAchievement

A badge class for microcredentials with the quality framework for professionals HBO/WO or MBO and all metadata conform the EU recommendations.

ExtraCurricularAchievement

A badge class for acquired knowledge and skills that are not part of the curriculum of a recognized education program. This badge class is not associated with ECTS/SBU.
